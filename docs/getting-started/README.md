# Getting Started with Polkadot Cookbook

Welcome! This section will help you get up and running with the Polkadot Cookbook.

## What is Polkadot Cookbook?

The Polkadot Cookbook is a collection of tested, production-ready code examples (recipes) for building on Polkadot. Each recipe demonstrates how to accomplish specific tasks using the Polkadot SDK.
