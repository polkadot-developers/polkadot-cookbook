// Quick Start

Up and running in minutes

Install the CLI, pick a recipe, and start building.

Step 01

Install the CLI

One command to install the dot CLI tool.

$ curl -fsSL https://raw.githubusercontent.com/polkadot-developers/polkadot-cookbook/master/install.sh | bash

Step 02

Run a recipe

Clone a recipe harness and run the tests to verify it works.

$ cd recipes/contracts/contracts-example $ npm ci # Install harness deps $ npm test # Clone, build & test
